## Synopsis

Complete keyboard shortcut reference for Microsoft Edge on Windows covering favorites, address bar, tabs, windows, navigation, developer tools, and PDF controls.

## Syntax

Shortcuts use standard modifier keys: Ctrl, Shift, Alt, and function keys (F1-F12).

## Parameters/Flags

- `Ctrl`: primary modifier for most shortcuts
- `Shift`: secondary modifier, often reverses direction (e.g., Ctrl+Tab vs Ctrl+Shift+Tab)
- `Alt`: used for menu access and navigation shortcuts
- `F1-F12`: function keys for DevTools, help, caret browsing, and Immersive Reader

|----------|--------|
